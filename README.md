Tech Stack: MERN

A website for colleges to register and host hackathons. Roles: Student, College, Volunteer, Company, Admin

## Features:

Common Features:

1. User authentication - (email otp verification)
2. Browse there profile
3. Give feedback 

### Student

1. Create teams (reach out to others with desired tech stack)
2. Hackathon registration
3. Contact volunteers via chat
4. Idea submission 

### College

1. Add volunteers
2. Browse companies and request sponsorship
3. Organize a hackathon ( give details, overview and timeline )

### Company

1. Approve or deny sponsorship requests from colleges
2. Suggest budget to any collage which is actively organize hackathon.

### Volunteer

1. Answer questions from students (hackathon participants)

```
Admin
```
